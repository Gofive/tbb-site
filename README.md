# Tbb-site Build By [Gatsby](https://www.gatsbyjs.com/) And [Material-ui](https://material-ui.com/zh/)

## How to run

```sh
npm install gatsby-cli -g
npm install
gatsby develop -- develop
gatsby build -- publish to public folder
```

## How to new a page

```sh
cd src/pages
touch newpage.js
/newpage  will be available
```
